About Medication Tracker

Built for treatment continuity, clinician review, and practical daily tracking.

Purpose

Medication Tracker helps record doses, symptom check-ins, and medication changes in one place.

Who it is for

This website is designed for personal use with optional read-only sharing for psychiatrist review.

How it supports appointments

Consult summaries surface current medications, recent changes, trends, and open questions before each appointment.

This tool supports tracking and clinician discussion. It does not replace professional medical advice.
